Frequently Asked Questions

Getting started

How do I install Quarto?
Installation is not required.
Download the stock quarto here or use it as a template.
Multiple quartos can exist on one machine without conflicts.
Can I build my site without icons, an email, a license, ...?
Yes. Delete the relevant option(s) or leave them blank.
See docs/options for details.
Why is requirements.txt blank?
Quarto has no requirements, only extras.
This file is used by the kitchen script for development and testing.
requirements.txt can be safely deleted.
What are the Dockerfile and kitchen files?
These files are used to build containers for development and testing.
Both files can be safely deleted if you don't use Docker.

Publishing a website

How do I publish a website?
Upload your proof folder to a web host.
Be careful not to upload anything private!
This site uses Neocities, a nonprofit which offers free and low-cost hosting.
Do I have to use my host's domain name?
Not if you register your own.
Many web hosts will register domains for an annual fee.
Already-registered domains are sometimes offered for sale.
llanfairpwllgwyngyllgogerychwyrndrobwllllantysiliogogogochuchaf.com is for sale.
A hosting service is telling me to buy a bunch of stuff. Do I need it?
You do not need frameworks, plugins, databases, server-side scripts, nor template languages to publish a website.

Features

Shall I compare thee to a static website generator?
Quartos are more lazy and more transparent.
Quartos are made to be customized, not configured.
Quarto code is stored in the same folder as site content.
Here are some website builders with richer feature sets:
  • Go: Hugo
  • JavaScript: Hexo, VuePress, Metalsmith, Eleventy
  • Python: MkDocs, Lektor, Pelican, Nikola, Hyde
  • Ruby: Middleman, Jekyll, Nanoc
Do quartos build dynamic websites?
No, but you could modify the Quire class to act as a server-side HTML generator and serve pages with something like bottle or flask or tornado.
Why doesn't Quarto have {x}?
Quarto is designed to minimize dependencies.
Quarto favors speed and simplicity over completeness.
If you really want to add a feature, DIY.
Where are all the context menus and modal windows?
I click to bury pop-ups, not engage them.

Purpose

Is Quarto a web development framework?
No. Quartos avoid inversion of control as much as possible.
Why does the Quarto icon have cat ears?
Quarto is lazy. It uses cached properties and generators to avoid schlemieling.
Quarto is small. It has < 500 lines of code, mostly in one file.
Quarto spends much of its time concatenating strings.
