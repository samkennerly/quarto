Chrome
View > Developer > View Source (Ctrl+U, ⌥⌘U)
View > Developer > Developer Tools (Ctrl+Shift+I, ⌥⌘I)
Firefox
Tools > Web Developer > Page Source (Ctrl+U, ⌘U)
Tools > Web Developer > Toggle Tools (Ctrl+Shift+I, ⌥⌘I)

help wanted

cascading stylist

Want to show off your CSS?
  1. Fork this repo and make your own stylesheet(s) in the style folder.
  2. Run quarto/apply style/your_style_name to apply your style.
  3. View the example pages on your machine to test your style.
  4. Create a pull request and/or publish it yourself.
Rules:
  1. Nobody should have to install anything to use your style.
  2. Try not to inflate the repo with files that are bigger than a galaxy.
  3. Styles can include non-CSS files like LICENSE. Quarto ignores them.
  4. If you use someone else's work, give credit and respect any licenses.
Quarto's default styles were made by a CSS rookie: me. Surely you can do better!
