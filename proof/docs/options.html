Options files

Options files are optional. See below for a complete list of all options.

The keys have values for those who JSON.

Page options are answers to questions like:

  • Who wrote this page?
  • Where are the stylesheet(s) and favicon?
  • Which icons should be in the #icons section?
  • Do you want to reveal your email to the entire internet?

Each page's options (if any) must be in a valid JSON file.

What's in a name?

If the path to a page is:

ready/examples/adrift.html

then that page's options (if any) must be in a file named:

ready/examples/adrift.json

Sites are merriest when they have a home.

  • The ready folder must have a page named index.html or ...
  • index.md, index.rst, index.htm, or any other suffix except .json.
  • If index.json exists, then its values are defaults for any missing options.
  • If a page has no options file, then all its values will be filled with default values.

Complete Options of Quarto

base
The canonical URL for your site with the path component removed.
"https://quarto.neocities.org/"
copyright
Copyright statement, if any.
"© Sam Kennerly 2019."
email
Email address to display, if any.
"samkennerly@gmail.com"
favicon
Icon file. Can be an absolute URL or relative path from home page.
"favicon.ico"
homelink
Label for link to home page in <nav> menu.
"Quarto"
icons
List of [alt text, image source, link URL] for #icons section.
Image sources can be absolute URLs and/or relative paths from home page.
[
  [ "GitHub", "media/icons/github.svg", "https://github.com/samkennerly" ],
  [ "Twitter", "media/icons/twitter.png", "https://twitter.com/data_lore/" ]
]
javascripts
List of script sources, if any.
Sources can be absolute URLs and/or relative paths from home page.
[
  "https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js",
  "src/javascript/annoyEveryoneWith-IntrusiveDialogBoxes.js"
]
license
Source and text of license, if any.
Source can be an absolute URL or relative path from home page.
[
  "https://creativecommons.org/licenses/by/4.0",
  "Licensed under a CC BY 4.0 license."
]
meta
{ name: content } mapping for <meta> tags, if any.
{
  "author": "Wm. Shakespeare",
  "description": "In Scotland, no one can hear you scream."
}
nextlink
Label for link to next page, if any.
"fwd ⇢"
prevlink
Label for link to previous page, if any.
"⇠ rev"
qlink
Label for link to quarto repository, if any.
"Built by a Quarto."
styles
Sources of stylesheet(s) for this page.
Sources can be absolute URLs and/or relative paths from home page.
[
  "https://www.w3schools.com/html/styles.css",
  "style.css"
]
title
Page <title> element.
"Macbeth II: Toil and Trouble"
