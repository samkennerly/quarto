Build websites with Python.

Quartos build websites. (This site is one.) Scroll through these stanzas to see how it's done.

Copy this quarto.

A quarto is four folders in a fifth folder:

  • proof is a proof copy of a static website.
  • quire is a Python package which builds the proof.
  • ready stores HTML (or Markdown) pages and JSON options files.
  • style stores CSS files. Use the four stock styles or make your own.

Download a ZIP or generate a repo from the original stock quarto:

github.com/samkennerly/quarto

Choose any name you want for your first quarto. Mine is badquarto.

Install extras?

Python 3.6+ is required. Some features require extras:

  • Tidy 5+ to clean HTML files with quire/clean
  • Mistune to build pages from Markdown files
  • Bottle to run a web server under construction
  • ImageMagick to build galleries under construction

Quartos do not need to be installed. To remove a quarto, delete it.

Rewrite the pages.

Any plain text editor can read and write pages, options, styles, and code.

Remove the stock pages.
Delete everything in the ready folder except these files:
ready/index.html
ready/index.json
Any change you might regret can be undone with git reset.
Write a new page.
Create ready/bad.html with this text:
<h1>Bad Quarto</h1>

<p>To be, or not to be?<br>Aye, there's the point.</p>

<figure>
  <img alt="Quarto logo" src="media/quarto.svg" width="100%">
  <figcaption>
    This page was not built by a
    <a href="https://github.com/samkennerly/quarto">quarto<a>.
  </figcaption>
</figure>
The image src is a relative path to a file in the proof folder.
Edit the home options.
Replace ready/index.json with this text:
{
  "base": "https://quarto.test/",
  "email": "",
  "homelink": "Bad Quarto",
  "meta": {
    "author": "Wm. Shakespeare",
    "description": "But, src! What links in yonder website break?",
  },
  "nextlink": "fore",
  "prevlink": "aft",
  "qlink": "Fouled by a quarto.",
  "styles": ["style.css"],
  "title": "Quarto test website."
}
Options are optional. If you don't want it, delete it or leave it blank.
Add some page options.
Create ready/bad.json with this text:
{
  "meta": {
    "description": "No is the status of your GET request."
  },
  "title": "The Tragedy of Errors",
}
Page options files are optional. Home options are used to fill missing values.

Build your new quarto.

Open a terminal and cd to your quarto.

quire/delete
Delete all pages from the proof folder:
quire/delete html
This script will not delete any icons, stylesheets, or cat pictures.
quire/clean
Clean and overwrite HTML files in the ready folder:
quire/clean
Keep your foul papers in another folder if you want to preserve them.
quire/build
Build and overwrite pages in the proof folder:
quire/build
This script uses each ready page as the main element of a proof page.
quire/apply
Apply a style from the style folder:
quire/apply style/celestial
This script finds CSS files recursively and cats them into one file.

Open proof/index.html in a browser to behold your brave new quarto.
